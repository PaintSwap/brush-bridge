# Fantom Sonic Tester

Try the speed of the new Fantom FVM

### Live at https://sonic.paintswap.app

## Development

Prerequisites:

* Node 18

Run development server:

```bash
yarn install
yarn dev
```

Build:

```bash
yarn build
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
