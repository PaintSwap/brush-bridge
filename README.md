## Development

Prerequisites:

* Node 18

Run development server:

Copy .env.sample to .env.local and add wallet connect project ID

```bash
yarn install
yarn dev
```

Build:

```bash
yarn build
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
